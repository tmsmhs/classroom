classroomの一日分の通知を一つのページで表示させるウェブアプリ（名称未定、命名権は作成者に帰属させる）

製作理由
クラスルームの通知は現状、各クラスルームのページに分散して通知される。しかしこれでは全体の通知を把握することは難しいため、
一つのページに集中させるウェブアプリを作成したいと考えた。

初期アイデア
classroomの通知は、classroomの各ページ、及びGmailに届く。そのうちGmailはGoogleAppsScriptでのデータ取得が容易であるため、
GoogleAppsScriptを使用してデータを集め、表示させる。
すべての通知を表示するのではGmailと大差がないため、教科ごとに分類ができるなどの差別化を図る。
